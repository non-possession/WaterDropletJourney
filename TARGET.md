# 《水的旅程》项目目标

## 项目一句话

《水的旅程》是一部使用 Godot 2D 制作的章节式诗性叙事探索游戏。玩家将陪伴一滴水在相遇、附着、离别、受纳、消散之中回归大海，体验“流动、关系、变化与归去”。

## 核心创作原则

- 弱目标，强感受。
- 以环境交互为骨架，以形态变化为玩法，以章节式体验承载情绪。
- 不是硬核平台跳跃，不是纯文字互动叙事，不是传统冒险解谜堆砌。
- 玩家不是在“读故事”，而是在“经历一种存在方式”。
- 追求的不是狭义上的“好不好玩”，而是玩家是否愿意持续陪伴这滴水，并在操作中感受它的处境、情绪和变化。

## 体验定位

本作适合定义为：

**2D 横向 / 轻纵深的诗性叙事探索游戏**

体验关键词：

- 有移动与操作
- 有环境反馈
- 有轻度谜题
- 有状态变化
- 有很强的视听表达
- 不被挑战性主导

目标感应保持存在，但避免任务列表化。玩家感受到的是“去往那里、穿过这一段、陪某种关系走完一程”，而不是完成系统化任务。

## 核心交互语言

本作的核心不是战斗、挑战或机关破解，而是以下交互语言：

- 流
- 停
- 聚
- 散
- 变
- 依附
- 渗透
- 离开
- 汇入

这些交互必须始终服务于主题表达：水在被世界改变的同时，也留下微小影响。

## 主要可玩状态

玩家控制的水滴将经历多种存在状态，并因状态变化获得不同的交互方式。

建议首批重点状态：

- 液态：可流动、滑移、穿过狭缝。
- 冰态：更稳定，可承重、滑行、碎裂、刺穿薄层。
- 雾态：可上升、扩散、穿越部分障碍。
- 附着态：可停驻在叶片、毛发、表面边缘。
- 渗透态：可进入木头、土壤、植物内部。
- 被吸收态：移动受限，但能感知身体或生命内部空间。

状态变化不是炫技，而是情绪与处境的游戏化表达。

## 章节结构

整体采用“大旅程 + 小关系章节”的组织方式。每一章都必须同时具备：

- 一个明确的情绪目的
- 一个明确的操作体验目的

### 1. 冰雪章

情绪关键词：

- 固化
- 稳定
- 束缚
- 离开

操作关键词：

- 滑行
- 冻结
- 碎裂
- 融化

### 2. 草木章

情绪关键词：

- 滋养
- 停驻
- 依恋
- 放手

操作关键词：

- 附着
- 渗透
- 沿叶脉流动
- 在给予中失去部分自己

### 3. 牛与小牛章

情绪关键词：

- 污浊
- 温暖
- 被吞纳
- 生命内部

操作关键词：

- 狭窄空间中的缓慢移动
- 脉动与挤压中的推进
- 对“脏”与“温”的重新感知

### 4. 眼泪章

情绪关键词：

- 情绪
- 失去
- 被带走
- 汇入更大之流

操作关键词：

- 减少主动控制
- 顺情绪轨迹下滑
- 汇流

## 关卡与交互设计原则

- 每个操作都要与世界发生关系。
- 环境不是背景板，而是会回应水的对象。
- 玩家不应“征服世界”，而是“在世界中被改变”。
- 谜题应服务于“水的处境”，而不是服务于解谜密度。
- 不使用高压失败惩罚破坏情绪连续性。

环境反馈示例：

- 石头让水分流
- 草叶让水停驻并压弯叶片
- 木头允许渗入并留下湿痕
- 风让雾态偏移
- 寒冷让边缘结晶
- 生物将水卷入生命循环

## 视听风格

### 视觉

- 2D 手绘 / 数字绘本感
- 柔和边缘与氛围光
- 不是纯像素，也不过度写实
- 留白、色块与层次气氛非常重要
- 水滴具有轻微拟人性，但不过分卡通
- 尽量少用明显表情，多用形态、亮度与运动传达情绪

### 镜头

- 以近中景为主
- 偶尔拉远以强调水与世界的尺度关系
- 关键时刻拉近，让玩家感受水体状态变化

### 声音

声音是本作的核心表达层之一，不是陪衬。

重点声音维度：

- 滴落
- 吸附
- 蒸腾
- 冰裂
- 渗透
- 被吞入
- 被挤压
- 顺叶滑落
- 眼泪下坠
- 汇入河流

环境声应重新定义同一种水：

- 冰雪里：更脆、更空
- 林地里：更柔、更近
- 动物体内：更闷、更温、更黏
- 河流里：更开阔、更连续

## 原型阶段开发目标

当前 Godot 原型应优先验证以下内容：

1. 水滴移动是否具有“流动感”，而不是普通角色跑跳感。
2. 液态 / 冰态 / 雾态等基础形态切换是否足够清晰可感。
3. 环境对不同状态是否能给出有意味的反馈。
4. 镜头、色彩、UI、音效是否能支撑“低压操作、强氛围”的体验。
5. 单个章节是否能同时成立情绪目标与操作目标。

## 当前项目范围

当前仓库阶段目标不是做完整内容，而是先建立：

- 清晰的项目愿景文档
- 可运行的 Godot 2D 项目
- 一个最小可玩原型场景
- 后续扩展章节、状态系统与环境交互的结构基础

## 成功标准

如果玩家在一小段原型体验后产生以下感受，则方向正确：

- “这一段我真的陪它走过了。”
- “我能感觉到它不是在闯关，而是在经历变化。”
- “它没有说很多，但我感到了。”
