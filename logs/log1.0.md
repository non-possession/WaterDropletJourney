# log1.0

## 第一版开发目标

目标：完成《水的旅程》第一章开场原型，一个小而完整、可游玩的高山冰雪开场切片。

这一版重点不是做长篇内容，而是先让玩家在 2 到 4 分钟内体验到：

1. 高山世界的辽阔、寒冷、静谧。
2. 同样来自水，却已经分化成不同状态。
3. 主角在群体中被看见，具有稳定而微弱的“自我”。
4. “回归大海”的召唤以画面和镜头而非大量文字被感知。
5. 玩家亲手带主角迈出离开的第一步。

## 第一版范围

只做以下内容：

- 一个高山冰雪开场场景
- 6 镜头开场分镜
- 主角基础待机与轻移动
- 少量背景中的“兄弟姐妹”水态表现
- 一个玩家接管后的简短可操作片段

当前不做：

- 完整世界观展开
- 长篇流程与多章节切换
- 重解谜或高压平台挑战
- 复杂音频系统
- 完整形态系统

## 模块拆分

### 模块 A：叙事镜头原型

- 搭建 6 个镜头节点
- 完成从宏观世界到主角个体，再到玩家接管控制的过渡
- 使用 UI 文本做少量镜头提示

### 模块 B：场景与分层

- 远景：天空、远山、东方暖光
- 中景：雪坡、冰面、山体、留白区域
- 近景：轻微雾层、风雪感、主角周围微弱碎光

### 模块 C：主角表现

- 接入现有 idle sprite
- 让待机有轻微呼吸感
- 让移动有轻盈、试探、克制的前行感
- 保持主角在视觉上比背景水体更稳定、更可被看见

### 模块 D：群体水态表现

- 冰：更静，更硬
- 雪：更轻，更散
- 想离开的同伴：轻微聚拢并朝东方倾向
- 留下者：保持环境状态

### 模块 E：占位资源与缺失资源管理

- 当前缺失的山体、雪坡、冰面、雾、风雪等资源暂时用程序化色块/多边形/透明层占位
- 将需要补做的 sprite / 美术 / 音频资源结构化记录

## 可直接发给大模型的素材请求模板

以下内容可以直接复制给大模型，让它帮助生成或规划素材。

### 1. 雪山远景背景

请为一款 2D、诗性叙事、冷色调的独立游戏生成一张横向远景背景图。内容是高山山脉、雪线、薄雾、寒冷空气感，整体轻灵、克制、留白，不要厚重描边，不要过度饱和，不要卡通冒险风。画面左中右需要有层次分明的远山轮廓，天空偏冷蓝，东方远处可以有极微弱暖光，暗示大海方向，但不要直接出现海。适合 Godot 2D 横向分层场景使用，分辨率建议至少 2560x1440，带轻微空气透视感。

### 2. 中景雪坡 / 山体拼接图

请为一款 2D 诗性叙事游戏生成中景雪坡与高山山体拼接素材，风格轻灵、冷色、半透明空气感，不要写实照片风，不要厚描边。需要可用于 Godot 横向关卡原型的雪坡、冰面、山体边缘拼接块，适合拼接成高山冰雪场景。材质重点是雪的轻、冰的透、山体的寒冷与高远。

### 3. 冰面地表贴图

请生成 2D 冰面地表贴图，适合横向叙事探索游戏。要求：冷蓝色、半透明、轻微裂纹与反光，不能太花，不要把冰做成厚重奇幻水晶地板，要保留留白和轻盈感，适合作为可行走区域的贴图。

### 4. 风雪粒子贴图

请生成用于 2D 游戏粒子系统的小型风雪粒子贴图。需要轻、细、稀疏、透明，像高山中被风带动的细雪与冰晶，不要暴风雪灾难片效果，不要太密。适合 Godot 粒子系统使用，透明背景。

### 5. 雾层 / 空气透视贴图

请生成 2D 横向游戏用的薄雾层贴图，透明背景，冷色调，轻微流动感，像高山空气中的薄雾和湿润寒气。整体要克制、留白，不要浓雾遮挡画面。

### 6. 群体小水灵 sprite

请为一款 2D 诗性叙事游戏生成几种“群体中的小水灵” sprite。它们来自同样的水，但没有主角那样稳定的内亮核。需要分别体现：停留者、随风者、准备离开者。整体应柔边、半透明、轻微发光、带生命感，但不要过度卡通化，不要像宠物吉祥物。透明背景，适合 Godot 2D 使用。

### 7. 主角轻移动动画

请为一滴具有微弱稳定内亮核的水灵设计 2D 轻移动循环动画。主角不是普通水滴，也不是重卡通角色。移动时应显得轻盈、试探、小心翼翼，带有“主动向前流动”的意志。需要透明背景 sprite 动画帧，建议 6 到 8 帧，横向移动时略有前倾、拉伸、轻微拖尾，但不能像高速冲刺特效。

### 8. 同伴轻微聚拢 / 转向东方动画

请为几滴准备离开高山、回归大海的水灵设计一个简短群体动画：它们先轻微靠近，再整体朝东方远望或偏转。不要对白，不要夸张情绪，用位置、角度、亮度变化传达“短暂连接与共同方向”。

### 10. 群体小水灵正式 sprite

请为一款 2D 诗性叙事游戏生成一组“群体中的小水灵”正式 sprite，透明背景，适合 Godot。需要至少三类：

- 停留者：更静、更弱亮核、边缘更稳定
- 准备离开者：轻微向前倾，有一点方向感和更清楚的内亮
- 随风 / 雪化者：更散、更轻、更接近环境状态

整体要求：

- 与主角同一审美体系
- 柔边、半透明、轻微发光
- 不要做成宠物吉祥物
- 不要过度卡通
- 不要把群体个体做得比主角更醒目

### 9. 最低限度音频包

请为一款 2D 高山冰雪、诗性叙事游戏列出第一章开场最小音频包清单。目标是只做最少但有效的声音：高山风、轻冰雪风切、主角待机水声、主角轻移动水声、镜头交接管时的提示音。请描述每个声音的质感、时长建议和使用场景。
